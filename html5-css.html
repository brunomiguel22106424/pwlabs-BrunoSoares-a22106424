Introdução ao HTML5

O que é?

O HTML é uma linguagem usada para a publicação de conteúdo (texto, imagens, vídeos, áudio) na web. O HTML5 é a nova versão e um dos seus principais objetivos é facilitar a manipulação dos elementos, possibilitando o desenvolvedor modificar as características dos objetos de forma não intrusiva. O HTML5 fornece ferramentas para o CSS e o Javascript fazerem seu trabalho da melhor possível de forma que um web site ou aplicação continue leve e funcional.

Elementos Semânticos Elementos Usados Explicação
header ✓ Usei no cabeçalho.
nav ✓ Usei no menu de navegação.
main ✓ Usei para a parte principal do site.
article ✓ Usei para dividir o main em pedaços.
section ✓ Usei para dividir o article em secções.
aside ✓ Usei para as imagens.
footer ✓ Usei para o rodapé.

Introdução ao CSS

O que é?

CSS é a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código criado por linguagens como HTML, por exemplo. De forma prática, ela funciona como uma camada de personalização ao conteúdo visível.

Elementos Semânticos Elementos Usados Explicação
universal ✓ O seletor universal aplica o estilo a todos os elementos da página. Usei para a fonte de letra.
tipo ✓ O seletor de tipo usa o nome do marcador a estilizar. Usei para estilizar os elementos dos marcadores.
classe ✓ O seletor de classe associa um nome a um estilo. Usei para dar um estilo especifico a um elemento do marcador.
id ❌ O atributo id permite identificar univocamente um elemento do HTML. Não usei.
atributo ✓ Seletores que se aplicam a atributos de elementos. Usei para dar estilo a alguns inputs do Quizz.
pseudo-classe ✓ Seletor associado ao estado de um elemento. Usei para hiperligações
pseudo-elemento ✓ Identificam elementos em determinado estado, na estrutura do documento. Usei para personalizar a tabela
